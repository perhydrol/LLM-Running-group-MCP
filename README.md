# LLM-Running-group-MCP - D&D MCP 工具集

这是一个基于 Model Context Protocol (MCP) 构建的跑团辅助工具集。旨在为 LLM 提供掷骰子、地图导航和世界构建等能力，使其能更好地扮演地下城主 (DM) 或辅助玩家。

> 纯 vibe coding 项目，祝你玩的愉快 (- v -)

## 项目包含的模块

本项目包含两个主要组件：

### 1. DND Dice MCP (`dnd_dice_mcp`)
一个专注于“叙事体验”的掷骰子服务。
- **左偏随机性**：模拟 D20 骰子，但结果略微偏向高分（众数为 18），以减少玩家挫败感。
- **难度分级**：支持从“轻轻松松”到“几乎不可能成功”的 5 个难度等级。

### 2. DND Map MCP (`dnd_map_mcp`)
一个用于管理和导航 D&D 城市地图的服务。
- **图导航系统**：基于图论的路径查找，支持不同权重的路径（距离/危险度）。
- **丰富元数据**：支持定义城市的中心冲突、派系和剧情钩子。
- **隐藏路径**：支持可被探索发现的秘密路径。
- **城市构建技能**：包含 `dnd-city-builder` Skill，指导 LLM 生成符合规范的城市数据。

## 安装与使用

本项目推荐使用 `uv` 进行依赖管理。

### 前置要求
- Python 3.10+
- `uv` 包管理器

### 安装依赖
在项目根目录下运行：
```bash
uv sync
```

### 运行服务

**运行骰子服务:**
```bash
cd dnd_dice_mcp
uv run server.py
```

**运行地图服务:**
```bash
cd dnd_map_mcp
uv run server.py
```

支持标准 IO (默认)传输模式。详细文档请参考各子目录下的 `README.md`。
